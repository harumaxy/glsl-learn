== shaderを使い回す

Sprite に `ShaderMaterial` をつけて、そこに `Shader` スクリプトをつけている。

ShaderMaterial をリソースとして保存すれば、別のシーンのノードに同じ Shader を適用できる。 +
`ShaderParam` を独自に適用した状態で保存できる +
`.material`拡張子で保存するのがおすすめ
